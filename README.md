## ✨ Key Features
- **AI Background Removal:** Remove backgrounds from any image in seconds using integrated AI logic.
- **Professional Filters:** Includes various presets like Grayscale, Warm, Cool, Vintage, and more.
- **Cross-Platform:** Works perfectly on Web (PWA) and as an Android APK.
- **Secure Authentication:** User login and data management powered by **Firebase**.
- **Dark Mode UI:** Sleek, modern, and professional aesthetic for better user experience.
- **Real-time Editing:** Instant preview of adjustments and filters.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Custom Dark Theme), JavaScript
- **Backend/Auth:** Firebase (Authentication & Database)
- **Deployment:** Netlify
- **App Wrapper:** Capacitor / Webview for Android APK
